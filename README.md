## ohmyking zsh theme

Single-line prompt without extra information 

![ohmyking](theme.jpg)

## Installation

For oh-my-zsh users, place into ~/.oh-my-zsh/custom/themes the file ohmyking.zsh-theme and change to ZSH_THEME="ohmyking" in your ~/.zshrc file
